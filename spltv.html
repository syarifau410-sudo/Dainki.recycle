Sistem Persamaan Linear Tiga Variable
(SPLTV)

x = satu produk mug ecoprint
y = satu produk jam dari motherboard
z = satu produk kalender dari tutup botol

2x + 2y + z = 201.500 ... (1)
x + 2y + z = 171.500 ... (2)
x + y + 2z = 208.000 ... (3)

•Eliminasi x dari persamaan (1) dan (2):
(1) - (2) => x = 30.000

•Substitusikan x = 30.000 ke persamaan (2):
30.000 + 2y + z = 171.500
2y + z = 141.500 ... (4)

•Substitusikan x = 30.000 ke persamaan (3):
30.000 + y + 2z = 208.000
y + 2z = 178.000 ... (5)

•Eliminasi y dari persamaan (4) dan (5):
(4) - 2*(5) => -3z = -214.500 z = 71.500

•Substitusikan z = 71.500 ke persamaan (4):
2y + 71.500 = 141.500
2y = 70.000
y = 35.000


HP: { (30.000, 35.000, 71.500) }
